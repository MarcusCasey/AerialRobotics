# AerialRobotics
My work regarding aerial robotics, as well as computer vision detection for humans in crisis scenarios.

To execute program cd into ir_hog directory

Enter:

python IAR.py images


If you would like to add more images to be processed, simply add them in PNG form to the images folder. You are able to delete applied folder at any time; the program will re-create the folder based on the images folder.
